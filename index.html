Console object. In javascript, the console is an object which provides access to the browser debugging console. We can open a console in web browser by using: Ctrl + Shift + K for windows and Command + Option + K for Mac. The console object provides us with several different methods, like : log ().The Console has 2 main uses: viewing logged messages and running JavaScript. Web developers often log messages to the Console to make sure that their JavaScript is working as expected.
